## 保真方式裁定(已定 / 四轮实测逐 ref 定案)

deepcopy 源 element(`<w:tbl>` / `<w:p>`)整体搬入 assembled.docx,deepcopy 后对跨 part ref 做处理:
- tblStyle:按 w:name 映射 rebind(源 styleId → master 同名 styleId)。1.docx 的 tblStyle=12 (name="Table Grid") → master TableGrid。零损失(列宽在 tblGrid 内 deepcopy 自带,边框由 rebind 后的 TableGrid 提供)
- pStyle:同上 name-based rebind。pStyle=3 (name="heading 2") → master Heading2。零损失
- numId:strip + emit warning。依据:段落字面无编号前缀(纯装饰)+ 孤立单段(0 牵连)+ master 无多级 decimal 可复用(rebind 过重)。strip 丢自动编号前缀,标题语义由 pStyle rebind 保留
- run 级字体 / 加粗 / 颜色 / Heading 等:deepcopy element 自带(无需额外处理,本就在 `<w:p>`/`<w:r>` 内)

## 实现指令(Phase 2 据实参照)

通则:每处落盘前实测当前确切字面 + 行号(字面 anchor)。deepcopy/rebind 逻辑先读 b_mode_fill 现有 _handle_asset_lookup 结构 + python-docx element API,不自创。

### S1 — element deepcopy

替换 _handle_asset_lookup 里 `for para in src_doc.paragraphs: new_p.add_run(para.text)`:
改为遍历源 doc body 的 element(src_doc.element.body 的 `<w:p>` 和 `<w:tbl>` 顺序),对每个用 copy.deepcopy 拷贝 element,append 到目标 doc body。这样表格(含 tblGrid 列宽)、段落级样式、run 级字体、单元格 multi-p 全部 element 内自带。保留 V3-1 的溯源说明行(deepcopy 后追加"asset 来源"段,逻辑不变)。

### S2 — ref rebind/strip

deepcopy 后,对搬入的 element 做跨 part ref 处理:
- 扫源 docx styles.xml 建 {源 styleId → w:name} 映射;扫 master styles.xml 建 {w:name → master styleId} 映射;两映射串起来得 {源 styleId → master styleId}
- 对搬入 element 内的 `<w:tblStyle w:val>` / `<w:pStyle w:val>`,按映射改 val(name-based rebind);映射不到的(master 无同名)→ strip 该 ref + warning(降级回 master 默认)
- 对 `<w:numPr>`(含 numId):strip + emit warning("asset 注入丢失自动编号前缀,如需可手补编号字面")
- rebind/strip 是对 deepcopy 副本操作,不动源 docx

### S3 — fixture + 测试

用 1.docx 作 anchor。测试分两类断言:
- 可自动验(XML 断言,复用 check_font_safety 的 zipfile 解):注入后 assembled.docx 内 `<w:tbl>` count ≥1 / `<w:tr>` 9 / 每行 `<w:tc>` 7 / `<w:gridCol w:w>` 7 个列宽值 == 源值(551/965/1314/2268/993/992/1213)/ `<w:tblStyle w:val="TableGrid">`(rebind 成功)/ `<w:pStyle w:val="Heading2">`(rebind 成功)/ `<w:numId>` 0 命中(strip 成功)/ run 级 `<w:b>` 加粗保留
- 必人工目检(测试里标注,不假装自动验):用 Word/WPS 打开注入后 docx,目检表格边框完整不变形、列宽视觉正确、Heading2 标题样式对、跨 WPS 渲染无字体 fallback。测试输出明确打印"以下项需人工目检:..."提示
断言禁泛断言;列宽断言用确切 dxa 值对照
