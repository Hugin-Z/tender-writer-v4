## 2. 实现设计

### 2.1 --section-only flag + 文件名自动检测

加 argparse flag(默认 False)。优先级规则(从高到低):

1. 用户显式传 `--section-only` → `section_only=True`
2. 否则若 docx_path 文件名含 `tender_response` 且不含 `final_response` → `section_only=True`
3. 否则 → `section_only=False`(全项检查;包括 `final_response.docx` / 任意其他名)

**冲突场景**:文件名同时含 tender_response 和 final_response(实际不会出现,但保险)→ final_response 更具终端语义,按规则 3 走全项。

**没有"强制全项"flag**:用户想要全项,只需不传 `--section-only` 且文件名不含 tender_response。这避免引入 `--full / --no-section-only` 三态语义。

### 2.2 should_section_only 抽函数

把决策逻辑抽到独立小函数,便于单测:

```python
def should_section_only(docx_filename: str, explicit_flag: bool) -> bool:
    """决定本次检查是否走 section-only 模式。

    优先级:
    1. 用户显式 --section-only → True
    2. 文件名含 'tender_response' 且不含 'final_response' → True
    3. 否则 False(全项)
    """
    if explicit_flag:
        return True
    name = docx_filename.lower()
    if "tender_response" in name and "final_response" not in name:
        return True
    return False
```

### 2.3 check_format 加 section_only param

`check_format(docx_text, format_info, section_only=False)` 内,当 `section_only=True` 时跳过 `FORMAT_CHECKS` 关键词循环(封面 + 目录),保留 section margins + heading count 检查:

```python
def check_format(docx_text: str, format_info: dict, section_only: bool = False) -> list[str]:
    issues: list[str] = []
    # ... section margins 检查不变 ...
    
    if not section_only:
        # 仅在全项模式才检查封面/目录关键词
        normalized_doc = normalize_for_match(docx_text)
        for check_name, keywords in FORMAT_CHECKS.items():
            if not all(normalize_for_match(keyword) in normalized_doc for keyword in keywords):
                kw_list = ', '.join(keywords)
                issues.append(f'missing {check_name} keywords: {kw_list}')
    
    # ... heading count 检查不变 ...
    return issues
```

向后兼容:默认 `section_only=False`,旧调用方不传该参数也跑全项。

### 2.4 main() 串起来

```python
parser.add_argument("--section-only", action="store_true",
    help="V3-5:跳过封面/目录关键词检查(适用于 tender_response.docx 等"
         "分章节产物)。文件名含 tender_response 时自动启用,可显式覆盖。")
args = parser.parse_args()

# 决定模式
section_only = should_section_only(docx_path.name, args.section_only)
if section_only:
    print(f"[信息] section-only 模式(跳过封面/目录关键词检查)", file=sys.stderr)

# 传给 check_format
format_issues = check_format(docx_text, format_info, section_only=section_only)
```
