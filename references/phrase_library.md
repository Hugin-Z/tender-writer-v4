## 附录:话术回填规则

1. **每条话术注明出处**:从哪个项目、哪份标书、哪次中标后回填的,方便追溯。
2. **每条话术标注适用场景**:适用于哪类项目(数字乡村 / 智慧城市 / 政务系统 / 教育 / 医疗等)。
3. **避免空话套话**:回填时优先选择有具体数据、具体场景、具体动作的句子,删除"较好""完善""高效""先进"等无信息量的形容词。
4. **优先回填中标项目的金句**:中标项目的话术经过了市场验证,优先级高于落标项目。
5. **季度归档一次**:每季度由 skill 维护人统一整理一次,避免重复和过时。
