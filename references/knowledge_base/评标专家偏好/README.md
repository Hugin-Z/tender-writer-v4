# 评标专家偏好

## 用途

记录不同行业、不同类型评标专家的关注点、偏好、扣分点。AI 在阶段 3(提纲)和阶段 4(正文)撰写时会扫描本目录中与当前项目相关的内容并吸收。

> ⚠️ 本目录的内容**只用于学习参考**,不会被复制到标书正文。但它会影响 AI 的应答策略——例如某行业评委特别关注某类指标时,AI 会在对应章节加重笔墨。

## 目录约定

每份 .md 都遵循 knowledge_base 通用 frontmatter:

```yaml
---
source_company: <来源公司 id>
company_type: own | partner | reference
review_status: pending | approved
source_file: <原始文件名,如:住建类项目复盘记录_2024.docx>
ingest_date: <YYYY-MM-DD>
---
```

文件名建议:`<行业>_<偏好类别>.md`,例如:
- `住建类项目常见关注点.md`
- `教育信息化项目常见关注点.md`
- `医疗信息化项目常见关注点.md`
- `数字乡村项目常见关注点.md`
