# own_default · 资质清单(索引)

> 本文件是 own_default 公司下所有资质的索引。摄入流程会自动追加新条目到下面的表格中。
> 同时每份资质都有一个独立的 `<资质名>.md` 详情文件遵循 `资质schema.md`。

## 状态说明

| 字段 | 含义 |
|---|---|
| review_status | `pending` = 刚摄入未审 / `approved` = 已审通过可引用 |
| 有效期至 | YYYY-MM-DD,临期(< 30 天)或过期会被阶段 5 终审标红 |

## 摄入说明

- 把资质证书的扫描件 PDF / 图片 / docx 放入 `_inbox/` 目录
- 在 Claude Code 中说"处理 own_default 的资质 inbox"
